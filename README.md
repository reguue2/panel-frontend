# Panel Frontend

Este repositorio contiene el frontend de un panel diseñado para contactar y gestionar la comunicación con negocios a través de la API de Meta. Este proyecto fue desarrollado por necesidad para facilitar la conexión con empresas mediante una interfaz gráfica alojada en **Render**.

## Características

- **Integración con la API de Meta**: Permite la comunicación con negocios directamente desde la plataforma.
- **Interfaz moderna**: Cuenta con un diseño sencillo y amigable para interactuar con las funciones clave.
- **Despliegue en Render**: Configurado para ser ejecutado en Render, garantizando una implementación ágil y escalable.

## Requisitos

Antes de comenzar, asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) (Versión 16 o superior)
- Un navegador web moderno para utilizar la interfaz
- Una cuenta activa en Render para desplegar el proyecto

## Instalación

1. Clona este repositorio:

   ```bash
   git clone https://github.com/reguue2/panel-frontend.git
   ```

2. Dirígete al directorio del proyecto:

   ```bash
   cd panel-frontend
   ```

3. Instala las dependencias necesarias:

   ```bash
   npm install
   ```

4. Configura las variables de entorno necesarias para la conexión con la API de Meta.

## Uso

1. Ejecuta la aplicación en modo de desarrollo:

   ```bash
   npm start
   ```

2. Abre tu navegador en: [http://localhost:3000](http://localhost:3000)

## Despliegue en Render

Para desplegar el proyecto en **Render**, sigue estos pasos:

1. Conecta tu repositorio en la plataforma de Render.
2. Configura el entorno necesario, especificando las variables para la API de Meta.
3. Inicia el despliegue y Render se encargará de la configuración automática.
